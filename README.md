# react-dashboard

This project is a React dashboard application that displays various statistics related to light usage. The dashboard includes cards that show the number of hours lights are on and off, total hours for the day, and future totals for the week and month.

## Features

- **Dashboard**: Aggregates and displays statistics in a user-friendly format.
- **Cards**: Individual components that present statistics in a visually appealing card layout.
- **Custom Hooks**: Manages state and data fetching for statistics.
- **TypeScript**: Provides type safety and better development experience.
- **Utility Functions**: Contains calculations for totals and averages.

## Installation

To get started with the project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd react-dashboard
npm install
```

## Usage

To run the application in development mode, use:

```bash
npm start
```

The application will be available at `http://localhost:3000`.
